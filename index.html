PROYECTS

PacMan

For my first project, I developed a Java-based Pac-Man clone designed to run entirely within the console. Its standout feature is the absence of predefined maps. I implemented an algorithm that procedurally generates a random maze for each session, ensuring a valid path is always available.

Controls are mapped to WASD keys, with real-time collision detection and game state management handled via matrix logic. The mechanics feature ghosts with random movement patterns and a dynamic difficulty system, where the challenge scales up or down in real-time based on the specific items consumed. The design prioritizes immediate 'arcade-style' replayability, with scoring tracked exclusively during the active session.

Restaurant Management

Restaurant Management is a Java-based desktop application designed for the comprehensive administration of a restaurant or bar.

It enables the management of users, tables, orders, and products, along with invoice generation and daily sales summaries, all operated through an organized Graphical User Interface (GUI).

Freezer Management

Development of a Full-Stack web application for comprehensive freezer and pantry management, aimed at reducing food waste.

The architecture features a lightweight Frontend built with native web standards (HTML5, CSS3, and Vanilla JavaScript) to ensure optimal performance without client-side framework dependencies. This interfaces with a RESTful Backend developed in Node.js and Express, while data persistence (users, products, expiration dates) is handled by a PostgreSQL relational database.
